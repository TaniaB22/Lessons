Сприсок компанд Git

$ git add . - Индексирует указанный файл для последующего коммита

$ git commit -m "[сообщение с описанием]" - Фиксирует проиндексированные изменения и сохраняет их в историю версий

$ git checkout -b "[название ветки]" - Создает новую ветку

$ git branch - Список именованных веток коммитов с указанием выбранной ветки

$ git branch -d [имя ветки] - Удаляет выбранную ветку

$ git push -u origin [ветка] - Загружает все изменения локальной ветки в удалённый репозиторий
